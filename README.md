# Rust Kernel for Jupyter

## Precondition

Rustcが必要です。

/* 動作確認はrustc 1.13.0 にて行いました。

## how to install

以下を実行してください。

```
git clone https://github.com/m0riiiii/jupyter-rust-kernel.git
cd jupyter-rust-kernel.git
sh install.sh
```
